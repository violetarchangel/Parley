## About This Project

First up, this game/engine was written by guided AI. I am not a coder, I'm a frustrated gamer. That means the code probably has quirks, inefficiencies, or outright weirdness that a human wouldn't have written. 
I'm admitting this upfront: **I am not a professional programmer, and I know AI makes mistakes.** The code is probably a shitshow, and I don't know enough about it to notice.

If you find them, and know more than a robot, please dig in and fix them. This project belongs to the community.

## Why This Exists

Over the years, there have been a few interactive fiction and chat-story platforms. They were closed-source, and often locked key features behind paywalls. I wrote stories for a couple of them, and now my stories are in the wind. They're all gone or abandoned now, servers shut down, user's submitted content, including my own, lost.

This is an attempt to do the opposite: **decentralized, open, and not-for-profit.**  No enshittification. Just a tool that exists for people who want to tell stories.

## Your Stories Are Yours

The license above applies to this *engine* (the code that runs the stories), **not to the stories you write with it**.

The story files (`*.js`) that are generated when you export a story are **your original content**. They are not covered by CC BY-NC or any other restriction from this project. You can:

- Sell your stories commercially
- License them however you want (closed source, proprietary, CC0, whatever)
- Delete them, modify them, distribute them — it's your work
- Use them in paid games, books, or other projects, whatever.

The only requirement is that **the engine itself remains free and non-commercial**. You can't sell *this software* or wrap it in a paid service, but you can absolutely sell the stories you create *with* it.

Think of it like a word processor: Microsoft doesn't own your novel, and this project doesn't own your interactive fiction.
